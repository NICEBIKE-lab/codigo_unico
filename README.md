# codigo_unico
Revisa codigo unico en filas

## Uso

1. Instala dependencias:

	```bash
	pip install -r requirements.txt
	```

2. Coloca un archivo Excel de entrada en la raíz con uno de estos nombres:
	- `archivo_entrada.xlsx`
	- `input.xlsx`
	- `data.xlsx`

3. Ejecuta:

	```bash
	python app.py
	```

El resultado se guarda como `archivo_unificado.xlsx`.
